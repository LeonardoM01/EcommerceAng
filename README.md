# EcommerceAng
Projeto de Ecommerce simples para aplicar pratica em Angular, API, .NET e Testes Unitários.

### Tecnologias Utilizadas
 - .NET 6.0;
 - API REST
 - Angular 18;
 - Sem Permanencia de Dados.

### Configuração do Ambiente (Pré-requisitos)
 - .NET SDK (6.0)
 - Runtime do ASP.NET Core 6.0
 - Node.js
 - AngularCLI

### Instalação
 - Clone o repositório
 - Acesse a pasta da API 
 > cd /Ecommerce.Api
 - Restaure dependências
 > dotnet restore
 - Execute a API
 > dotnet run
 - Acesse a pasta do FronteEnd
 > cd /Ecommerce.WEB
 - Instale as dependências
 > npm install
 - Suba o angular
 > npm start
